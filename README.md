# 🀄 麻雀アシスタント

麻雀を覚えたい人のための学習・対戦支援ツールです。ブラウザだけで動作し、インストール不要で使えます。

🔗 **[ツールを開く](https://haruto25252.github.io/mahjong-assistant/)**

---

## 主な機能

### 🎮 ゲームプレイ
- **四麻 / 三麻** の切り替え対応
- CPU 対戦（3人または4人）
- リーチ・ツモアガリ・ロン判定
- ドラ表示・符翻計算・自動得点表示

### 🤖 CPU アルゴリズム選択
| モード | 説明 |
|--------|------|
| ツモ切り | 完全ランダム。何も考えず捨てる |
| 役狙い | シャンテン数を最小化して向かっていく |
| 守備型 | 他プレイヤーの捨て牌を参考に安全牌を優先 |
| 攻撃型 | 早アガリ重視 |

### 💡 捨て牌アシスタント
- 捨てるべき牌の候補をシャンテン数順にリスト表示
- 各候補の「詳細」ボタンで以下を確認できる
  - 捨てた後の手牌形
  - シャンテン数（テンパイ・アガリ形も表示）
  - テンパイ時の待ち牌一覧
  - 成立する役

### 📖 役一覧
- 一翻〜役満まで全役を一覧表示
- 各役の翻数・説明を確認できる

### 🔍 テキスト検索
- 画面左上の 🔍 ボタンで検索モードをON
- 気になる用語をタップ／クリックするだけで意味・読み方を即表示
- 東家・シャンテン・放銃・符など 60 種類以上の麻雀用語辞書を内蔵
- Google / Wikipedia への外部検索リンクも常時表示

### 📱 レスポンシブ対応
- 起動時にデスクトップ / スマートフォンのレイアウトを選択
- スマホレイアウトでは下部タブバーで「ゲーム・得点・候補・ログ」を切り替え
- 設定パネルからいつでもレイアウトを変更可能

### 💾 ゲーム状態の自動保存
- 5秒ごとにブラウザへ自動保存（localStorage）
- 外部サイトで調べ物をして戻ってきても進行状況が保持される

---

## 使い方

1. [ツールを開く](https://haruto25252.github.io/mahjong-assistant/)
2. レイアウト（デスクトップ / スマホ）を選択
3. 自動でゲームが始まります
4. 手牌の牌をクリック／タップして捨てる
5. 右パネル（スマホはタブ）で捨て牌候補を確認

## 技術スタック

- **HTML / CSS / JavaScript** のみ（フレームワーク不使用）
- **Google Fonts** （Noto Serif JP・Zen Antique）
- **localStorage** によるゲーム状態の永続化
- 単一HTMLファイル構成 → そのままGitHub Pagesで動作
